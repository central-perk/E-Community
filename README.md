E-Community For ECD

```
1. install `node.js` `mongodb`
2. run mongod
3. `$ make install` Install the dependencies of Nodeclub 
4. `$ make test` 
5. `$ node app.js`
6. visit `localhost:3000`
7. done!
```

## More information

Unit test

```bash
$ make test
```

Coverage tests

```bash
$ make test-cov
```

```
mongod --dbpath ./db
```
